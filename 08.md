# Chapter 8. Zero-Copy와 Scatter-Gather I/O
게임 서버의 성능을 좌우하는 중요한 요소 중 하나는 메모리 복사 오버헤드다. 패킷을 송수신할 때마다 발생하는 불필요한 메모리 복사는 CPU 사이클을 낭비하고 캐시 효율을 저하시킨다. 이 장에서는 메모리 복사를 최소화하는 Zero-Copy 기법과, WSASend/WSARecv의 강력한 기능인 Scatter-Gather I/O를 활용하여 성능을 극대화하는 방법을 학습한다.

고성능 게임 서버는 단순히 기능이 동작하는 것을 넘어서, 제한된 하드웨어 자원으로 최대한 많은 동시 접속자를 처리해야 한다. 불필요한 메모리 복사를 제거하는 것은 이러한 목표를 달성하는 핵심 기법이다.
  

## 8.1 메모리 복사 오버헤드 분석
전통적인 방식의 패킷 송신 과정을 살펴보면 여러 번의 메모리 복사가 발생한다는 것을 알 수 있다.

**전형적인 패킷 송신 과정**

```
┌──────────────┐
│ 패킷 생성    │  ← 복사 1: 데이터를 패킷 버퍼로 복사
└──────┬───────┘
       │
┌──────▼───────┐
│ 송신 큐 추가 │  ← 복사 2: 패킷을 송신 큐로 복사
└──────┬───────┘
       │
┌──────▼───────┐
│ 링 버퍼 복사 │  ← 복사 3: 큐에서 링 버퍼로 복사
└──────┬───────┘
       │
┌──────▼───────┐
│ WSASend 호출 │  ← 복사 4: 링 버퍼에서 커널 버퍼로 복사
└──────┬───────┘
       │
┌──────▼───────┐
│ 네트워크 전송│
└──────────────┘
```

각 복사 단계에서 CPU는 메모리를 읽고 쓰는 작업을 수행한다. 특히 큰 패킷이나 많은 패킷을 처리할 때 이 오버헤드는 상당하다.

**메모리 복사의 비용**

메모리 복사가 왜 비싼지 구체적으로 살펴보자.

```cpp
// 1MB 데이터를 복사하는 시간 측정
void MeasureCopyCost()
{
    constexpr size_t SIZE = 1024 * 1024;  // 1MB
    std::vector<char> src(SIZE);
    std::vector<char> dst(SIZE);
    
    auto start = std::chrono::high_resolution_clock::now();
    
    // 100번 복사
    for (int i = 0; i < 100; ++i)
    {
        memcpy(dst.data(), src.data(), SIZE);
    }
    
    auto end = std::chrono::high_resolution_clock::now();
    auto duration = std::chrono::duration_cast<std::chrono::microseconds>(end - start);
    
    LOG_INFO("100MB 복사 시간: {}μs", duration.count());
    // 일반적으로 수십 밀리초 소요
}
```

1MB를 복사하는데 수백 마이크로초가 걸린다면, 초당 1만 개의 1KB 패킷을 처리하는 서버는 복사에만 수십 밀리초를 소비한다. 이는 전체 프레임 시간의 상당 부분을 차지할 수 있다.

**캐시 미스 문제**

메모리 복사는 CPU 캐시를 오염시킨다. 복사된 데이터가 캐시를 채우면서 정작 필요한 게임 로직의 데이터가 캐시에서 밀려나간다.

```
L1 캐시 (32KB, 가장 빠름)
  ↓ 미스
L2 캐시 (256KB, 빠름)
  ↓ 미스
L3 캐시 (8MB, 보통)
  ↓ 미스
메인 메모리 (느림, 100배 이상 느림)
```

불필요한 복사로 인해 캐시 미스가 증가하면 전체 시스템 성능이 크게 저하된다.

**기존 구현의 문제점**

Chapter 7까지 구현한 코드를 분석해보자.

```cpp
// 문제가 있는 코드
void Session::Send(PacketPtr packet)
{
    // 복사 1: shared_ptr 복사 (제어 블록 참조)
    sendQueue_.push(packet);
    
    RegisterSend();
}

bool Session::RegisterSend()
{
    // 복사 2: 큐에서 패킷들을 벡터로 복사
    std::vector<PacketPtr> packets;
    PacketPtr packet;
    while (sendQueue_.try_pop(packet))
    {
        packets.push_back(packet);
    }
    
    // 복사 3: 패킷 내용을 링 버퍼로 복사
    sendBuffer_.Clear();
    for (auto& pkt : packets)
    {
        sendBuffer_.Write(pkt->GetBuffer(), pkt->GetSize());  // memcpy!
    }
    
    // 복사 4: WSASend (링 버퍼 → 커널 버퍼)
    sendWsaBuf_.buf = sendBuffer_.GetReadBuffer();
    sendWsaBuf_.len = static_cast<ULONG>(sendBuffer_.GetDataSize());
    WSASend(socket_, &sendWsaBuf_, 1, ...);
}
```

이 코드에서 패킷 데이터는 최소 3번 복사된다 (shared_ptr은 포인터 복사이므로 제외). 패킷이 1KB라면 3KB의 메모리 복사가 발생한다.

**최적화 목표**

우리의 목표는 다음과 같다:

1. 패킷 데이터를 직접 WSASend에 전달하여 중간 복사 제거
2. 여러 패킷을 한 번의 시스템 콜로 전송
3. 수신 시에도 커널 버퍼에서 직접 패킷 파싱

이를 달성하기 위해 Scatter-Gather I/O를 사용한다.
  

## 8.2 Scatter-Gather I/O 이해
Scatter-Gather I/O는 여러 개의 불연속적인 메모리 버퍼를 하나의 I/O 작업으로 처리하는 기법이다. Winsock의 WSASend와 WSARecv는 이를 네이티브로 지원한다.

**Scatter I/O (분산 읽기)**

수신 시 하나의 네트워크 스트림을 여러 버퍼로 분산하여 읽는다.

```
네트워크 스트림: [Header][Body1][Body2]
                   ↓      ↓      ↓
분산 저장:       [Buf1] [Buf2] [Buf3]
```

**Gather I/O (집중 쓰기)**

송신 시 여러 버퍼의 데이터를 하나의 스트림으로 모아서 전송한다.

```
여러 버퍼:     [Packet1] [Packet2] [Packet3]
                  ↓         ↓         ↓
네트워크 전송: [Packet1Packet2Packet3]
```

**WSABUF 배열**

WSASend와 WSARecv는 WSABUF 구조체의 배열을 받을 수 있다.

```cpp
typedef struct _WSABUF {
    ULONG len;    // 버퍼 크기
    CHAR* buf;    // 버퍼 포인터
} WSABUF, *LPWSABUF;

// 여러 버퍼를 한 번에 전송
WSABUF buffers[3];
buffers[0].buf = packet1->GetBuffer();
buffers[0].len = packet1->GetSize();
buffers[1].buf = packet2->GetBuffer();
buffers[1].len = packet2->GetSize();
buffers[2].buf = packet3->GetBuffer();
buffers[2].len = packet3->GetSize();

WSASend(socket, buffers, 3, &bytesSent, 0, &overlapped, nullptr);
```

이렇게 하면 3개의 패킷이 메모리 복사 없이 한 번의 시스템 콜로 전송된다. 커널이 직접 각 버퍼에서 데이터를 읽어 네트워크로 전송한다.

**장점**

1. **메모리 복사 제거**: 중간 버퍼가 필요 없다
2. **시스템 콜 감소**: 여러 패킷을 한 번에 전송
3. **캐시 효율**: 불필요한 메모리 접근 감소
4. **구현 단순화**: 링 버퍼 관리 불필요

**주의사항**

WSABUF 배열의 각 버퍼는 WSASend가 완료될 때까지 유효해야 한다. 비동기 I/O이므로 버퍼가 조기에 해제되거나 수정되면 안 된다.

```cpp
// 잘못된 예
void BadSend()
{
    WSABUF buf;
    char localBuffer[1024];  // 스택 변수!
    buf.buf = localBuffer;
    buf.len = 1024;
    
    WSASend(socket, &buf, 1, nullptr, 0, &overlapped, nullptr);
    
    // 함수가 리턴되면 localBuffer가 무효화됨
    // WSASend가 완료되기 전에 스택이 해제되어 크래시!
}

// 올바른 예
void GoodSend()
{
    auto buffer = std::make_shared<std::vector<char>>(1024);
    
    WSABUF buf;
    buf.buf = buffer->data();
    buf.len = static_cast<ULONG>(buffer->size());
    
    // IOContext에 buffer의 shared_ptr 저장
    ioContext->buffer = buffer;  // 수명 보장
    
    WSASend(socket, &buf, 1, nullptr, 0, &ioContext->overlapped, nullptr);
}
```

## 8.3 WSABUF 배열을 이용한 다중 버퍼 전송

이제 실제로 WSABUF 배열을 사용하여 여러 패킷을 한 번에 전송하는 구조를 구현한다.

**SendContext 설계**

WSASend가 완료될 때까지 패킷들의 수명을 보장해야 한다. 이를 위해 전용 컨텍스트를 만든다.

```cpp
struct SendContext
{
    OVERLAPPED overlapped;
    SessionPtr session;
    
    // 전송할 패킷들 (수명 보장)
    std::vector<PacketPtr> packets;
    
    // WSABUF 배열
    std::vector<WSABUF> wsaBufs;
    
    SendContext()
    {
        ZeroMemory(&overlapped, sizeof(OVERLAPPED));
    }
    
    void AddPacket(PacketPtr packet)
    {
        WSABUF buf;
        buf.buf = packet->GetBuffer();
        buf.len = static_cast<ULONG>(packet->GetSize());
        
        packets.push_back(packet);
        wsaBufs.push_back(buf);
    }
    
    WSABUF* GetBufferArray()
    {
        return wsaBufs.data();
    }
    
    DWORD GetBufferCount() const
    {
        return static_cast<DWORD>(wsaBufs.size());
    }
};
```

**Session의 송신 개선**

```cpp
class Session
{
public:
    void Send(PacketPtr packet)
    {
        // 송신 큐에 추가 (복사 없음, shared_ptr만 이동)
        sendQueue_.push(std::move(packet));
        
        // 송신 시도
        RegisterSend();
    }
    
    bool RegisterSend()
    {
        // 이미 송신 중이면 중복 방지
        bool expected = false;
        if (!isSending_.compare_exchange_strong(expected, true))
            return false;
        
        // SendContext 생성
        auto sendContext = new SendContext();
        sendContext->session = shared_from_this();
        
        // 큐에서 패킷들 가져오기 (최대 64개)
        PacketPtr packet;
        while (sendQueue_.try_pop(packet) && sendContext->packets.size() < 64)
        {
            sendContext->AddPacket(std::move(packet));
        }
        
        // 가져올 패킷이 없으면 중단
        if (sendContext->packets.empty())
        {
            delete sendContext;
            isSending_.store(false);
            return false;
        }
        
        // WSASend 호출 (Gather I/O)
        DWORD bytesSent = 0;
        int result = WSASend(
            socket_,
            sendContext->GetBufferArray(),
            sendContext->GetBufferCount(),
            &bytesSent,
            0,
            &sendContext->overlapped,
            nullptr
        );
        
        if (result == SOCKET_ERROR)
        {
            int error = WSAGetLastError();
            if (error != WSA_IO_PENDING)
            {
                LOG_ERROR("WSASend 실패: Error={}", error);
                delete sendContext;
                isSending_.store(false);
                return false;
            }
        }
        
        return true;
    }
    
    void OnSend(SendContext* sendContext, DWORD bytesTransferred)
    {
        // 송신 완료
        LOG_DEBUG("송신 완료: PacketCount={}, Bytes={}",
                 sendContext->packets.size(), bytesTransferred);
        
        // SendContext 정리 (패킷들의 shared_ptr 자동 해제)
        delete sendContext;
        
        // 송신 플래그 해제
        isSending_.store(false);
        
        // 큐에 남은 패킷이 있으면 계속 송신
        if (!sendQueue_.empty())
        {
            RegisterSend();
        }
    }
    
private:
    std::atomic<bool> isSending_{false};
    concurrency::concurrent_queue<PacketPtr> sendQueue_;
};
```

**워커 스레드 수정**

```cpp
void IOCPServer::WorkerThread()
{
    while (running_)
    {
        DWORD bytesTransferred = 0;
        ULONG_PTR completionKey = 0;
        OVERLAPPED* overlapped = nullptr;
        
        GetQueuedCompletionStatus(iocpHandle_, &bytesTransferred,
                                 &completionKey, &overlapped, INFINITE);
        
        // SendContext인지 IOContext인지 구분 필요
        // 간단한 방법: OVERLAPPED 구조체의 첫 필드를 타입으로 사용
        
        // 또는 별도 식별자 추가
        auto baseContext = reinterpret_cast<BaseContext*>(overlapped);
        
        if (baseContext->type == ContextType::SEND)
        {
            auto sendContext = static_cast<SendContext*>(baseContext);
            sendContext->session->OnSend(sendContext, bytesTransferred);
        }
        else if (baseContext->type == ContextType::RECV)
        {
            auto ioContext = static_cast<IOContext*>(baseContext);
            ioContext->session->OnRecv(bytesTransferred);
            delete ioContext;
        }
    }
}
```

**메모리 복사 비교**

기존 방식:
```
Packet → Queue → Vector → RingBuffer → Kernel
 (복사1)  (복사2)   (복사3)     (복사4)
```

Scatter-Gather 방식:
```
Packet → Queue → SendContext → Kernel
 (포인터)  (포인터)        (DMA)
```

실제 메모리 복사가 거의 사라졌다. shared_ptr은 제어 블록 포인터만 복사하므로 오버헤드가 매우 작다.
  

## 8.4 Zero-Copy 기법 적용
Scatter-Gather I/O로 송신은 최적화했지만, 수신 쪽도 개선할 수 있다. Zero-Copy의 핵심은 데이터를 제자리에서 처리하는 것이다.

**수신 버퍼 직접 파싱**

기존에는 수신 버퍼에서 패킷을 추출할 때 새로운 Packet 객체로 복사했다.

```cpp
// 기존 방식 (복사 발생)
std::vector<PacketPtr> Session::ExtractPackets()
{
    std::vector<PacketPtr> packets;
    
    while (recvBuffer_.GetDataSize() >= PACKET_HEADER_SIZE)
    {
        uint16_t packetSize = 0;
        recvBuffer_.Peek(&packetSize, sizeof(packetSize));
        
        if (recvBuffer_.GetDataSize() < packetSize)
            break;
        
        // 복사 발생!
        auto packet = std::make_shared<Packet>(packetSize);
        recvBuffer_.Read(packet->GetBuffer(), packetSize);
        
        packets.push_back(packet);
    }
    
    return packets;
}
```

**Zero-Copy 방식**

버퍼에서 직접 파싱하고, 필요한 경우에만 복사한다.

```cpp
// Zero-Copy 방식
void Session::ProcessReceivedData(DWORD bytesTransferred)
{
    recvBuffer_.MoveWritePos(bytesTransferred);
    
    while (recvBuffer_.GetDataSize() >= PACKET_HEADER_SIZE)
    {
        // 헤더만 Peek (복사 없음)
        const char* bufferPtr = recvBuffer_.GetReadBuffer();
        uint16_t packetSize = *reinterpret_cast<const uint16_t*>(bufferPtr);
        uint16_t packetID = *reinterpret_cast<const uint16_t*>(bufferPtr + 2);
        
        // 크기 검증
        if (packetSize < PACKET_HEADER_SIZE || packetSize > MAX_PACKET_SIZE)
        {
            Disconnect();
            break;
        }
        
        // 완전한 패킷 확인
        if (recvBuffer_.GetDataSize() < packetSize)
            break;
        
        // 버퍼에서 직접 파싱 (복사 없음!)
        PacketReader reader(bufferPtr, packetSize);
        
        // 핸들러 실행
        auto handler = PacketDispatcher::Instance().GetHandler(packetID);
        if (handler)
        {
            handler->Handle(shared_from_this(), reader);
        }
        
        // 읽은 만큼 버퍼 전진
        recvBuffer_.MoveReadPos(packetSize);
    }
}
```

**PacketReader 개선**

PacketReader가 버퍼를 복사하지 않고 직접 참조하도록 수정한다.

```cpp
class PacketReader
{
public:
    // 버퍼를 복사하지 않고 포인터만 저장
    PacketReader(const char* buffer, size_t size)
        : buffer_(buffer)
        , size_(size)
        , pos_(PACKET_HEADER_SIZE)  // 헤더는 건너뜀
    {
    }
    
    template<typename T>
    T Read()
    {
        if (pos_ + sizeof(T) > size_)
        {
            hasError_ = true;
            return T{};
        }
        
        // 직접 메모리에서 읽기 (복사 없음)
        T value = *reinterpret_cast<const T*>(buffer_ + pos_);
        pos_ += sizeof(T);
        return value;
    }
    
    std::string ReadString()
    {
        uint16_t length = Read<uint16_t>();
        if (hasError_ || pos_ + length > size_)
        {
            hasError_ = true;
            return "";
        }
        
        // 여기서만 복사 발생 (불가피)
        std::string str(buffer_ + pos_, length);
        pos_ += length;
        return str;
    }
    
    // Zero-Copy 문자열 뷰 (C++17)
    std::string_view ReadStringView()
    {
        uint16_t length = Read<uint16_t>();
        if (hasError_ || pos_ + length > size_)
        {
            hasError_ = true;
            return "";
        }
        
        // 복사 없음!
        std::string_view view(buffer_ + pos_, length);
        pos_ += length;
        return view;
    }
    
private:
    const char* buffer_;  // 복사 없이 참조만
    size_t size_;
    size_t pos_;
    bool hasError_ = false;
};
```

**주의사항**

Zero-Copy 방식은 버퍼가 유효한 동안만 안전하다. 비동기로 처리하거나 저장해야 한다면 복사가 필요하다.

```cpp
// 위험한 예
void LoginHandler::Handle(SessionPtr session, PacketReader& reader)
{
    auto userID = reader.ReadStringView();  // Zero-Copy 뷰
    
    // 비동기 DB 조회 시작
    dbThread_.Post([session, userID]() {  // ← 위험!
        // userID는 이미 무효화된 메모리를 가리킬 수 있음
        auto result = DB::QueryUser(userID);
        // ...
    });
}

// 안전한 예
void LoginHandler::Handle(SessionPtr session, PacketReader& reader)
{
```

```cpp
#pragma once
#include <cstdint>
#include <string>
#include <string_view>

class PacketReader
{
public:
    PacketReader(const char* buffer, size_t size)
        : buffer_(buffer)
        , size_(size)
        , pos_(4)  // 헤더(Size + ID) 건너뜀
    {
    }
    
    template<typename T>
    T Read()
    {
        if (pos_ + sizeof(T) > size_)
        {
            hasError_ = true;
            return T{};
        }
        
        T value = *reinterpret_cast<const T*>(buffer_ + pos_);
        pos_ += sizeof(T);
        return value;
    }
    
    // 복사 버전 (필요시)
    std::string ReadString()
    {
        uint16_t length = Read<uint16_t>();
        if (hasError_ || pos_ + length > size_)
        {
            hasError_ = true;
            return "";
        }
        
        std::string str(buffer_ + pos_, length);
        pos_ += length;
        return str;
    }
    
    // Zero-Copy 버전 (C++17)
    std::string_view ReadStringView()
    {
        uint16_t length = Read<uint16_t>();
        if (hasError_ || pos_ + length > size_)
        {
            hasError_ = true;
            return "";
        }
        
        std::string_view view(buffer_ + pos_, length);
        pos_ += length;
        return view;
    }
    
    bool HasError() const { return hasError_; }
    size_t GetRemainingSize() const { return size_ - pos_; }
    
private:
    const char* buffer_;
    size_t size_;
    size_t pos_;
    bool hasError_ = false;
};
```

```cpp
bool Session::RegisterRecv()
{
    if (!IsConnected())
        return false;
    
    // 버퍼 정리 (읽은 데이터 제거)
    recvBuffer_.Compact();
    
    // IOContext 생성
    auto ioContext = new IOContext();
    ioContext->session = shared_from_this();
    
    // WSABUF 설정
    recvWsaBuf_.buf = recvBuffer_.GetWriteBuffer();
    recvWsaBuf_.len = static_cast<ULONG>(recvBuffer_.GetFreeSize());
    
    DWORD flags = 0;
    DWORD bytes = 0;
    
    int result = WSARecv(socket_, &recvWsaBuf_, 1, &bytes, &flags,
                        &ioContext->overlapped, nullptr);
    
    if (result == SOCKET_ERROR)
    {
        int error = WSAGetLastError();
        if (error != WSA_IO_PENDING)
        {
            LOG_ERROR("WSARecv 실패: SessionID={}, Error={}", sessionID_, error);
            delete ioContext;
            Disconnect();
            return false;
        }
    }
    
    return true;
}

void Session::OnRecv(DWORD bytesTransferred)
{
    if (bytesTransferred == 0)
    {
        Disconnect();
        return;
    }
    
    EnqueueJob([this, bytesTransferred]() {
        recvBuffer_.MoveWritePos(bytesTransferred);
        ProcessReceivedData();
        RegisterRecv();
    });
    
    TryProcessJobs();
}

void Session::ProcessReceivedData()
{
    while (recvBuffer_.GetDataSize() >= PACKET_HEADER_SIZE)
    {
        // Zero-Copy: 버퍼에서 직접 읽기
        const char* bufferPtr = recvBuffer_.GetReadBuffer();
        
        uint16_t packetSize = *reinterpret_cast<const uint16_t*>(bufferPtr);
        uint16_t packetID = *reinterpret_cast<const uint16_t*>(bufferPtr + 2);
        
        // 크기 검증
        if (packetSize < PACKET_HEADER_SIZE || packetSize > MAX_PACKET_SIZE)
        {
            LOG_ERROR("잘못된 패킷 크기: Size={}, SessionID={}", 
                     packetSize, sessionID_);
            Disconnect();
            break;
        }
        
        // 완전한 패킷 확인
        if (recvBuffer_.GetDataSize() < packetSize)
            break;
        
        // Zero-Copy 파싱
        PacketReader reader(bufferPtr, packetSize);
        
        // 핸들러 실행
        auto handler = PacketDispatcher::Instance().GetHandler(packetID);
        if (handler)
        {
            try
            {
                handler->Handle(shared_from_this(), reader);
            }
            catch (const std::exception& e)
            {
                LOG_ERROR("패킷 처리 예외: PacketID={}, SessionID={}, Error={}",
                         packetID, sessionID_, e.what());
            }
        }
        else
        {
            LOG_WARNING("등록되지 않은 패킷: PacketID={}, SessionID={}",
                       packetID, sessionID_);
        }
        
        // 읽은 만큼 전진
        recvBuffer_.MoveReadPos(packetSize);
    }
}

void Session::Send(PacketPtr packet)
{
    sendQueue_.push(std::move(packet));
    RegisterSend();
}

bool Session::RegisterSend()
{
    // 중복 송신 방지
    bool expected = false;
    if (!isSending_.compare_exchange_strong(expected, true))
        return false;
    
    // SendContext 생성
    auto sendContext = new SendContext();
    sendContext->session = shared_from_this();
    
    // 큐에서 패킷들 가져오기 (Scatter-Gather I/O)
    PacketPtr packet;
    while (sendQueue_.try_pop(packet) && 
           sendContext->packets.size() < MAX_SEND_BATCH)
    {
        sendContext->AddPacket(std::move(packet));
    }
    
    if (sendContext->packets.empty())
    {
        delete sendContext;
        isSending_.store(false);
        return false;
    }
    
    // Gather I/O: 여러 패킷을 한 번에 전송
    DWORD bytesSent = 0;
    int result = WSASend(
        socket_,
        sendContext->GetBufferArray(),
        sendContext->GetBufferCount(),
        &bytesSent,
        0,
        &sendContext->overlapped,
        nullptr
    );
    
    if (result == SOCKET_ERROR)
    {
        int error = WSAGetLastError();
        if (error != WSA_IO_PENDING)
        {
            LOG_ERROR("WSASend 실패: SessionID={}, Error={}", sessionID_, error);
            delete sendContext;
            isSending_.store(false);
            Disconnect();
            return false;
        }
    }
    
    return true;
}

{
    LOG_DEBUG("송신 완료: SessionID={}, PacketCount={}, Bytes={}",
             sessionID_, sendContext->packets.size(), bytesTransferred);

    delete sendContext;
    
    // 송신 플래그 해제
    isSending_.store(false);
    
    // 큐에 남은 패킷이 있으면 계속 송신

    {
        RegisterSend();
    }
```

```cpp
#pragma once
#include <vector>
#include <cstring>

class LinearBuffer
{
public:
    explicit LinearBuffer(size_t capacity)
        : buffer_(capacity)
        , capacity_(capacity)
    {
    }
    
    char* GetReadBuffer()
    {
        return buffer_.data() + readPos_;
    }
    
    char* GetWriteBuffer()
    {
        return buffer_.data() + writePos_;
    }
    
    size_t GetDataSize() const
    {
        return writePos_ - readPos_;
    }
    
    size_t GetFreeSize() const
    {
        return capacity_ - writePos_;
    }
    
    void MoveReadPos(size_t size)
    {
        readPos_ += size;
        if (readPos_ > capacity_)
            readPos_ = capacity_;
    }
    
    void MoveWritePos(size_t size)
    {
        writePos_ += size;
        if (writePos_ > capacity_)
            writePos_ = capacity_;
    }
    
    // 읽은 데이터 제거하고 앞으로 이동
    void Compact()
    {
        if (readPos_ == 0)
            return;
        
        size_t dataSize = GetDataSize();
        if (dataSize > 0)
        {
            memmove(buffer_.data(), buffer_.data() + readPos_, dataSize);
        }
        
        readPos_ = 0;
        writePos_ = dataSize;
    }
    
    void Clear()
    {
        readPos_ = 0;
        writePos_ = 0;
    }
    
private:
    std::vector<char> buffer_;
    size_t capacity_;
    size_t readPos_ = 0;
    size_t writePos_ = 0;
};
```

```cpp
void IOCPServer::WorkerThread()
{
    LOG_INFO("워커 스레드 시작: ThreadID={}", GetCurrentThreadId());
    
    while (running_)
    {
        DWORD bytesTransferred = 0;
        ULONG_PTR completionKey = 0;
        OVERLAPPED* overlapped = nullptr;
        
        BOOL result = GetQueuedCompletionStatus(
            iocpHandle_,
            &bytesTransferred,
            &completionKey,
            &overlapped,
            INFINITE
        );
        
        // 종료 신호
        if (completionKey == 0 && overlapped == nullptr)
            break;
        
        // BaseContext로 타입 확인
        auto baseContext = reinterpret_cast<BaseContext*>(overlapped);
        
        // I/O 실패 처리
        if (!result || bytesTransferred == 0)
        {
            if (baseContext->type == ContextType::RECV)
            {
                auto ioContext = static_cast<IOContext*>(baseContext);
                ioContext->session->Disconnect();
                delete ioContext;
            }
            else if (baseContext->type == ContextType::SEND)
            {
                auto sendContext = static_cast<SendContext*>(baseContext);
                sendContext->session->Disconnect();
                delete sendContext;
            }
            continue;
        }
        
        // 타입별 처리
        if (baseContext->type == ContextType::RECV)
        {
            auto ioContext = static_cast<IOContext*>(baseContext);
            ioContext->session->OnRecv(bytesTransferred);
            delete ioContext;
        }
        else if (baseContext->type == ContextType::SEND)
        {
            auto sendContext = static_cast<SendContext*>(baseContext);
            sendContext->session->OnSend(sendContext, bytesTransferred);
            // OnSend에서 delete됨
        }
    }
    
    LOG_INFO("워커 스레드 종료: ThreadID={}", GetCurrentThreadId());
}
```

이제 우리는 Zero-Copy와 Scatter-Gather I/O를 완전히 통합한 고성능 IOCP 게임 서버를 구축했다. 주요 최적화 포인트:

1. **Zero-Copy 수신**: 버퍼에서 직접 파싱, 불필요한 복사 제거
2. **Scatter-Gather 송신**: 여러 패킷을 한 번에 전송, 링 버퍼 불필요
3. **선형 버퍼**: Compact 연산으로 메모리 효율 유지
4. **배치 처리**: 최대 64개 패킷을 한 번의 시스템 콜로 전송

다음 챕터에서는 게임 서버의 필수 기능인 타이머와 하트비트 시스템을 구현한다.
